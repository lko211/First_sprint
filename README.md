# First_sprint
Краткая документация:
Веб-API для создания, чтения, обновления и удаления событий (CRUD). Данные хранятся в памяти приложения в течение одной сессии. 

Реализованы методы:
GET для получения всех событий. URL: /api/events
GET для получения события по ID. URL: /api/events/{id}
POST для создания нового события. URL: /api/events
PUT для обновления события по ID. URL: /api/events/{id}
DELETE для удаления события по ID. URL: /api/events/{id}

Для запуска приложения необходимо в CLI:
git clone клонировать репозиторий
cd first_sprint перейти в директорию проекта
dotnet build для сборки билда
dotnet run для запуска проекта
https://localhost:5001/swagger в строке браузера необходимо ввести URL
